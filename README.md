# ⚔️ Hero XP Challenge

Sistema de classificação de heróis baseado em pontos de experiência (XP), desenvolvido como primeiro desafio do curso de Lógica de Programação da [DIO.me](https://dio.me).

---

## 📌 Sobre o projeto

O programa recebe o nome e a quantidade de XP de um herói e retorna seu nível de classificação. A lógica utiliza um objeto de faixas percorrido com `for...in`, retornando o nível correspondente ao primeiro limite atingido.

| XP acumulado | Nível |
|---|---|
| Até 1.000 | Ferro |
| Até 2.000 | Bronze |
| Até 3.000 | Prata |
| Até 4.000 | Ouro |
| Até 6.000 | Platina |
| Até 8.000 | Ascendente |
| Até 10.000 | Imortal |
| Acima de 10.000 | Radiante |

**Exemplo de saída:**
```
Gabriel o Bárbaro adquiriu um total de 850 pontos de xp e atingiu o nível Ferro
```

## ▶️ Como executar

1. Clone o repositório:
```bash
git clone https://github.com/GabrieILima/HERO-XP-CHALLENGE.git
```
2. Abra o arquivo `PRIMEIRO DESAFIO.js` em qualquer editor (VS Code recomendado)
3. Execute no terminal com Node.js:
```bash
node "PRIMEIRO DESAFIO.js"
```
